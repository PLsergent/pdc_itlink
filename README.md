# Outil\_PDC ITLink

### Installation

Clone the repository :

```text
git clone https://github.com/PLsergent/pdc_itlink
```

Install dependencies :

```text
cd pdc_itlink/
pipenv install
```

Run the application :

```text
pipenv run python manage.py runserver
```

By default got to this url : [http://127.0.0.1:8000/pdc/](http://127.0.0.1:8000/pdc/)

Good to go !
